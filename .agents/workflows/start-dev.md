---
description: How to start the development server on the correct port (8080)
---
1. Ensure all dependencies are installed: `npm install`
// turbo
2. Start the Vite development server: `npm run dev`
3. Antigravity should automatically detect that the server is running on `http://localhost:8080`.
4. If you need Antigravity to verify the UI, ask it to "Open the home page at localhost:8080".
